### 🌦️ Weather Notifier App (Python GUI)  
 
 A simple and interactive **Weather Notifier Application** built using Python.  
This app allows users to enter a city name and get **real-time weather updates** with a **GUI interface** and **desktop notifications**.

## 🚀 Features

- 🌍 Get real-time weather data for any city  
- 🖥️ User-friendly GUI using Tkinter  
- 🔔 Desktop notifications using Plyer  
- ⚡ Fast API-based data fetching  
- ❌ Handles invalid city inputs  

---

## 🛠️ Tech Stack

- **Python**
- **Tkinter (GUI)**
- **Requests (API calls)**
- **Plyer (Notifications)**
- **Open-Meteo API (No API Key required)**

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/weather-notifier.git
cd weather-notifier

### 2️⃣ Install Dependencies
pip install requests plyer

### 3️⃣ Run the Application
python app.py
```



## 🧠 How It Works

 -User enters city name
 -App fetches coordinates using Geocoding API
 -Weather API returns live data
 -Data is displayed in GUI
 -Notification is shown on desktop

---
 User → Python App → API → Data → Notification
